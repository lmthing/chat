# lmthing.chat

This repository hosts the GitHub Pages deployment for **lmthing.chat** — the personal THING interface.

The source code lives in the [`chat/`](https://github.com/lmthing/lmthing/tree/main/chat) directory of the [lmthing monorepo](https://github.com/lmthing/lmthing).

## Deployment

The site is automatically built and deployed to GitHub Pages whenever a commit is pushed to `main` in the [lmthing/lmthing](https://github.com/lmthing/lmthing) repository. You can also trigger a deploy manually via the Actions tab.
